Past Projects

Cooking is cool

Stack

  • HTML5/CSS3
  • Javascript/Jquery
  • Node/Express/Mongoose
  • MongoDB
  • Yummly RESTful API

Description

Cooking is cool is an app that uses the Yummly API to allow the user to:

  • Search for recipes by keyword.
  • Select recipes to create a menu.
  • Create a shopping list for that menu.
  • Edit that shopping list for items the user already has in their pantry.
  • Cross items off the list as they are added to their shopping cart.
  • Provides link to the recipes.

This app follows MVC architecture. Model/persitance is provided by MongoDb. Control is implemented with Node.js/Express and View is standard HTML/CSS/Javascript.

YouTube Notes

Stack

  • HTML5/CSS3
  • Javascript/Jquery
  • YouTube RESTful API
  • Node/Express/Mongoose
  • MongoDB
  • React

Description

YouTube Notes allows the user to search for a YouTube video by keyword. The user can then select from a list of returns and view the video within the app. While viewing the video, the user can take notes and save them. The application allows the user to view and edit their previous notes.

This app follows MVC architecture. Model/persitance is provided by MongoDb. Control is implemented with Node.js/Express and View is React.

Technologies/Skills

I have developed projects in the M.E.R.N stack.

MERN is a set of Open Source components that together, provide an end-to-end framework for building dynamic web applications; starting from the top (code running in the browser) to the bottom (database). The stack is made up of:

  • MongoDB : Document database – used by your back-end application to store its data as JSON (JavaScript Object Notation) documents
  • Express: Back-end web application framework running on top of Node.js
  • React: Front-end web app framework; runs your JavaScript code in the user's browser, allowing your application UI to be dynamic
  • Node.js : JavaScript runtime environment – lets you implement your application back-end in JavaScript

A common theme in the MERN stack is JavaScript – every line of code you write can be in the same language.

In addition I have used Git/GitHub for version control. Mocha and Enzyme for testing. As well as common developement tools such as Brackets, Sublime, and Notepad++
